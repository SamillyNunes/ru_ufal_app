# RU UFAL App

A mobile project built for Hackaton in Aracomp 2021. Aims to gather feautures for UFAL Restaurant, including initially a menu food.

### Requirements
- Flutter
- Dart
See https://flutter.dev/docs/get-started/install for more details.
